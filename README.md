# Rental Management System (CRUD)

This is a simple Rental Management System built in Java, implementing full Create, Read, Update and Delete (CRUD) functionality. It allows users to manage rental records through a clean and easy-to-use console interface.

## Features
- Add new rental entries  
- Update existing rental records  
- Delete rental records  
- View all rental data  
- Console-based user interaction  
- Modular and beginner-friendly structure  

## Tech Stack
- Java (Core Java, OOP)
- Console-based UI

## Project Structure
src/
├── models/ # Entity classes
├── services/ # CRUD logic
├── controllers/ # Menu and user interaction
└── Main.java # Entry point
